# 🎨 Artist Studio

A premium digital art studio platform built with **Ruby on Rails 8**, **Hotwire**, and **Tailwind CSS v4**.

Artist Studio combines a private artwork management system with a public-facing portfolio experience, allowing artists to organize, showcase, and present their work through a modern gallery-inspired interface.

---

## ✨ Features

### 🖼️ Artwork Portfolio

* Public artist portfolio
* Responsive gallery layouts
* Artwork detail pages
* Clean and immersive viewing experience
* Optimized image presentation

### 🎨 Studio Management

* Private artwork management dashboard
* Create, update, and organize artworks
* Manage artwork metadata
* Streamlined content administration

### ⚡ Modern Rails Stack

* Ruby on Rails 8
* Hotwire (Turbo + Stimulus)
* Tailwind CSS v4
* Importmaps
* SQLite (development)

### 🎭 Design Philosophy

Inspired by modern art galleries and museum experiences:

* Minimalist presentation
* Elegant typography
* Content-first design
* Fast navigation
* Responsive across devices

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Saurabhkr001/Artist-Studio.git
cd Artist-Studio
```

### Install Dependencies

```bash
bundle install
```

### Setup Database

```bash
bin/rails db:create
bin/rails db:migrate
```

### Start the Application

```bash
bin/dev
```

Visit:

```text
http://localhost:3000
```
